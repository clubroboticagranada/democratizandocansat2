Democratizando CANSAT II. El Retorno

Cansat Fake

Trata de simular la recogida de datos en tiempo real de un lanzamiento en cohete de cansat.

Para ello con los datos de un lanzamiento real se limpian y se ponen en formato JSONL (es un formato JSON en multilineas) y se guardan en una tarjeta microSD. Cambiamos el programa de emisión para que analice y extraiga cada linea del fichero para mandarla via LoRa al receptor cada segundo. Por tanto en emisión no enviamos datos medidos sino los de un fichero de un lanzamiento ya realizado.

Pasos para realizar la simulación:

  1. Decargar el fichero JSON de datos e introducirlos en la microSD del emisor.
  2. Cargar programa a emisor.
  3. Cargar programa a receptor.
  4. Proceder como una operación de emisión / recepción normal de democratizando CANSAT.

Programa de emisión: https://www.steamakersblocks.com/web/project/2633435

Programa de recepción: https://www.steamakersblocks.com/web/project/2448593
