Tratamiento de datos csv

Tratamiento de datos en el ordenador

Una vez la placa con su receptor LoRa está enviando datos al PC, éste debe tener un programa lector de puerto serie, que permita grabar los datos en un fichero, para esta tarea en GNU/Linux proponemos usar Serial Port Terminal (GTKTerm). Además vamos a utilizar un graficador en tiempo real llamado KST, que recogerá los datos del fichero grabado por gtkterm y en tiempo real los graficará.

El proceso completo está explicado en el siguiente vídeo:

Como alternativa para ordenadores con sistema operativo Windows, podemos usar para hacer lecturas de puertos serie (en windows com) el programa putty, y para el graficado en tiempo real podemos usar igualmente KST, ya que es un programa multiplataforma con versiones para GNU/Linux, Windows y Mac OSX.

Tratamiento de datos con hoja de cálculo

El fichero grabado en el PC a través de comunicación serie (por usb) con la placa receptora LoRa, o el fichero que guarda la tarjeta microSD del emisor LoRa con los datos transmitidos, pueden ser analizados y graficados con una hoja de cálculo, como se muestra en el siguiente vídeo.
